# 영역전개 - 주술회전 카드 배틀 게임

주술회전(呪術廻戦)을 기반으로 한 전략 카드 배틀 리그 게임입니다.

## 게임 소개

6장의 카드로 크루를 구성하고, 시즌제 리그에서 AI 크루들과 대결하여 우승을 목표로 합니다. 정규시즌 14경기(홈/어웨이 2회전) 후 상위 4팀이 플레이오프에 진출하며, 5전 3선승제로 챔피언을 가립니다.

## 주요 기능

### 🎮 시즌 리그 시스템
- **정규시즌**: 7개 AI 크루와 홈/어웨이 2회씩 대결 (총 14경기)
- **플레이오프**: 상위 4팀 진출, 5전 3선승제
  - 준결승: 1위 vs 4위, 2위 vs 3위
  - 결승: 준결승 승자끼리 대결
- **시즌 히스토리**: 역대 시즌 기록 저장
- **통산 전적**: 크루별 상대 전적 추적

### 🃏 캐릭터 카드 (20장)
- **특급 (3장)**: 고죠 사토루, 료멘 스쿠나, 켄자쿠
- **1급 (7장)**: 후시구로 토지, 나나미 켄토, 죠고, 하나미, 초소, 오쿠츠 유타, 토도 아오이
- **준1급 (6장)**: 이타도리 유지, 후시구로 메구미, 마히토, 메이메이, 이누마키 토게, 젠인 마키
- **2급 (4장)**: 쿠기사키 노바라, 이노 타쿠마, 판다, 니시미야 모모

### ⚔️ 전투 시스템
- **HP 기반 턴제 전투**: HP가 0이 될 때까지 공격을 주고받음
- **선공 결정**: SPD가 높은 쪽이 먼저 공격
- **스킬 발동**: 확률/조건에 따라 스킬 자동 발동
- **속성 상성**: 6속성 상성 시스템 (결계 > 저주 > 육체 > 사거리 > 혼백 > 반전 > 결계)
- **경기장 효과**: 25개 경기장별 고유 효과 (4개 카테고리)

### 🔄 트레이드 시스템
- **등급 포인트제**:
  - 특급: 10pt
  - 1급: 5pt
  - 준1급: 3pt
  - 2급: 2pt
  - 준2급/3급: 1pt
- **트레이드 규칙**: 포인트 차이 ±1 이내 거래 가능
- **AI 평가 로직**: 특급/1급 카드는 거절 확률 높음
- **강제 트레이드**: 플레이어만 사용 가능 (제한 없음)

### 🏆 우승 보너스
플레이오프 우승 시 등급 제한 확장:
- 특급: 1장 → 2장
- 1급: 2장 → 3장

### 📊 카드 기록 시스템
- **통산 기록**: 전체 시즌 누적 기록
- **시즌별 기록**: 각 시즌 개별 기록
- **시즌 성적 탭**: 확장 통계 표시
  - 최대 연승, 입힌/받은 데미지
  - 라운드 MVP, 스킬 발동 횟수
  - 데미지 효율 계산
- **경기장별 전적**: 경기장별 승/패 기록
- **상대 카드별 전적**: 교전 상대별 승/패 기록

### 🏅 수상 시스템
시즌 종료 시 자동 선정:
- **MVP**: 기여도 최고 (승리×2 + 승률×20)
- **다승왕**: 시즌 최다 승리

### 📰 뉴스 피드
자동 생성되는 게임 내 뉴스:
- 경기 결과
- 연승/연패 기록
- 트레이드 소식
- 시즌 이벤트
- 플레이오프 소식

### 📚 술사 명부 (전체 카드 도감)
- 전체 20장 카드 열람
- 등급/속성 필터링
- 다양한 정렬 옵션 (이름, 등급, 승률, 스탯 등)
- 크루 소속 정보 표시

### 📈 개인 순위
- 시즌별/통산 카드 순위
- 승수, 승률, 총 경기 수 등 다양한 기준

### 🛠️ 성장 시스템
- 대전 결과에 따른 경험치 획득
- 레벨업 시 주성장/부성장 스탯 +2
- 업적 달성을 통한 아이템 해금
- 장비 아이템 장착 (카드당 2슬롯)

## 기술 스택

- **프레임워크**: React 18 + TypeScript
- **빌드 도구**: Vite
- **상태 관리**: Zustand (LocalStorage 영속성)
- **스타일링**: Tailwind CSS v4
- **애니메이션**: Framer Motion

## 게임 규칙

### 크루 구성
- 6장의 카드로 크루 구성
- **등급 제한** (기본):
  - 특급: 최대 1장
  - 1급: 최대 2장
  - 준1급 이하: 제한 없음
- 우승 시 등급 제한 확장

### 대전 진행
1. 5라운드 진행 (3점 선승제)
2. 매 라운드 랜덤 경기장 선정
3. 플레이어와 AI가 각각 카드 선택
4. SPD가 높은 쪽이 선공
5. HP가 0이 될 때까지 공격 교환
6. 스킬 효과 자동 적용

### 승리 조건
- **라운드 승리**: 상대 HP를 0으로 만들면 승리
- **경기 승리**: 먼저 3점을 획득하면 승리

### 시즌 흐름
1. **크루 선택**: 게임 시작 시 6장의 카드 선택
2. **정규시즌**: 7개 AI 크루와 홈/어웨이 대결 (14경기)
3. **플레이오프**: 상위 4팀 5전 3선승제
4. **시즌 종료**: 수상 선정, 우승 보너스 지급
5. **다음 시즌**: AI 크루 재배정, 새로운 시즌 시작
