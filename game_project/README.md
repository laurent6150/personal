# 주술회전 카드 게임 "영역전개" (Domain Expansion)

## 🎮 게임 개요

주술회전 세계관을 기반으로 한 전략 카드 대전 + 수집/성장형 웹 게임

### 핵심 특징
- **5장 크루 시스템**: 랜덤 5장으로 팀 구성
- **5판 3선승제**: 한 번 쓴 카드는 재사용 불가
- **6속성 상성**: 결계 > 저주 > 신체 > 원거리... 순환 구조
- **성장 시스템**: 레벨업 (최대 Lv.10) + 장비 장착 (2슬롯)
- **AI 대전**: 난이도 선택 (EASY/NORMAL/HARD)
- **전적 기록**: 카드별, 경기장별, 상대별 데이터 축적

## 🎯 속성 상성

| 속성 | 강함 (×1.5) | 약함 (×0.7) |
|------|-------------|-------------|
| 결계 🔮 | 저주, 변환 | 신체, 혼백 |
| 신체 💪 | 결계, 변환 | 저주, 원거리 |
| 저주 👁️ | 신체, 원거리 | 결계, 혼백 |
| 혼백 👻 | 결계, 저주 | 변환, 원거리 |
| 변환 🔥 | 혼백, 원거리 | 결계, 신체 |
| 원거리 🎯 | 신체, 혼백 | 저주, 변환 |

| 등급 | 덱 제한 | 예시 |
|------|---------|------|

| B (준1급) | 제한없음 | 이타도리, 메구미, 마히토 |
| C (2급) | 제한없음 | 노바라, 판다 |

## ⚔️ 데미지 공식

```
데미지 = (ATK × 속성배율 × CE배율 × 경기장보너스) - DEF

- 속성배율: 유리 1.5 / 동등 1.0 / 불리 0.7
- CE배율: 1 + (CE / 100)
- 최소 데미지: 1
```
