# Daily Insight Skill

사용자가 전달하는 내용을 분석하여 핵심 인사이트를 추출하고 체계적으로 정리합니다.

## 출력 형식

인사이트를 정리할 때 반드시 다음 형식을 따릅니다:

### 1. 핵심 키워드 (Keywords)
- 전달받은 내용에서 **3~5개의 핵심 키워드**를 추출
- 키워드는 해시태그 형식으로 표시: `#키워드1` `#키워드2` `#키워드3`

### 2. 한눈에 보기 (Summary)
- 전체 내용을 **2~3문장**으로 압축하여 요약
- 핵심 메시지와 맥락을 담아 작성
- "오늘의 핵심:" 으로 시작

### 3. 테마별 정리 (Themes)
- 내용을 **2~4개 테마**로 분류
- 각 테마당 **2~4개 bullet point**로 핵심 내용 정리
- 테마명은 `##` 헤더로 표시
- 각 bullet point는 간결하고 실행 가능한 형태로 작성

### 4. 실천 포인트 (Action Items)
- 내용에서 도출할 수 있는 **실천 가능한 항목** 1~3개 제시
- 체크박스 형식으로 표시: `- [ ]`

## 저장 규칙

- 파일 경로: `insights/YYYY-MM-DD.md`
- 같은 날짜에 여러 인사이트가 있으면 기존 파일에 `---` 구분선 후 추가

## 출력 템플릿

```markdown
# YYYY-MM-DD 인사이트

## 핵심 키워드
`#키워드1` `#키워드2` `#키워드3`

## 한눈에 보기
오늘의 핵심: [2~3문장 요약]

## 테마별 정리

### 테마 1
- 핵심 포인트 1
- 핵심 포인트 2

### 테마 2
- 핵심 포인트 1
- 핵심 포인트 2

## 실천 포인트
- [ ] 실천 항목 1
- [ ] 실천 항목 2
```

## 처리 지침

1. **본질 파악**: 표면적인 내용보다 숨겨진 의미와 패턴을 찾습니다
2. **연결성**: 다른 개념이나 이전 인사이트와의 연결점을 발견합니다
3. **실용성**: 바로 적용할 수 있는 형태로 정리합니다
4. **간결성**: 불필요한 수식어를 제거하고 핵심만 남깁니다
